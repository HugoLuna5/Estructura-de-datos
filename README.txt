Practica 1: 
Programa que calcula el promedio para un conjunto de datos de tres maneras diferentes que son:
Versión 1: Con datos predefinidos."
Versión 2: Con n datos ingresados por teclado"
Versión 3: con n datos implementado en un método

Practica 2:
Programa que permite elegir un archivo de texto (TXT) a leer/procesar que hace sus operaciones básicas: 
insertar, modificar, eliminar, ordenar, buscar.

Practica 3:
El Programa le permite elegir el archivo de texto (TXT) que  almacena n calificaciones parciales de una asignatura para m estudiantes, una vez almacenada esta información,  el programa calcula el promedio general por estudiante, el promedio del grupo, el promedio mínimo y máximo del grupo.

Practica 4:
Programa le permite  elegir el archivo de texto (TXT). Permite llevar un control de n calificaciones para m estudiantes y p carreras obtenidas desde un archivo de texto (TXT).El Programa calcula:
-el promedio general de cada estudiante
-el promedio por cada carrera
-el promedio mínimo y máximo de cada carrera
-el promedio mínimo y máximo de todas las carreras.
El estudiante de el archivo de texto (TXT) debe estar conformado por:
Nombre
iniciales de la materia\
iniciales de la carrera
las calificaciones a declarar.
Por ejemplo:
Juan FP ISC 85 93 88.6 79
